# Architecture Documents

This directory contains design decisions, architecture comparisons, and technical design documents.

## What Goes Here

- **ADRs** (Architecture Decision Records): Immutable records of technical decisions
- **Architecture Comparisons**: Multi-option analysis documents
- **Design Documents**: Component/system design specifications
- **Database Schemas**: Schema definitions and migration plans

## Current Documents

- `AUTH_COMPARISON.md` - Analysis of authentication approaches for OAuth token management

## ADR Numbering

Next ADR number: **001**

When creating ADRs:
1. Number sequentially (ADR_001, ADR_002, etc.)
2. Use format: `ADR_[number]_[short_title].md`
3. Never edit past ADRs (create new ones that supersede)

See `../PROJECT_DOCUMENTATION_GUIDE.md` for ADR template.
