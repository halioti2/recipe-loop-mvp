# Retrospectives

This directory contains post-mortems and retrospectives from major launches, incidents, and project milestones.

## What Goes Here

- **Project Retrospectives**: What went well/poorly after launches
- **Incident Post-Mortems**: Root cause analysis after outages
- **Quarterly Reviews**: Periodic reflection on progress

## Naming Convention

`YYYY_MM_[project_name]_retro.md`

Example: `2026_03_auth_migration_retro.md`

## Status

Currently empty. Create retros 1-2 weeks after major milestones.

## Why This Matters

Retrospectives help:
- Learn from mistakes
- Celebrate wins
- Improve processes
- Share knowledge
